# Carpeta para estudiantes — Codificación y escalamiento

## Objetivo
Desarrollar la guía de aprendizaje sobre preparación de datasets para modelos de inteligencia artificial, aplicando codificación de variables categóricas, escalamiento de variables numéricas, separación entrenamiento/prueba y documentación técnica.

## Orden recomendado de trabajo
1. Leer la guía de aprendizaje en `01_Guia_Aprendizaje`.
2. Revisar el dataset y el diccionario de variables en `02_Datos_Trabajo`.
3. Abrir el notebook plantilla en `03_Notebook_Plantilla`.
4. Desarrollar cada sección del notebook y guardar avances por sesión.
5. Elaborar el informe técnico usando la plantilla en `04_Plantillas_Entregables`.
6. Validar la entrega con la rúbrica y la lista de chequeo en `05_Evaluacion`.

## Entregables finales
- Notebook desarrollado `.ipynb`.
- Dataset de entrenamiento preparado `.csv`.
- Dataset de prueba preparado `.csv`.
- Informe técnico en PDF o Word.
- Socialización final del proceso.

## Advertencia académica
Este paquete no contiene la solución completa. El propósito es que cada aprendiz construya el proceso paso a paso y justifique sus decisiones técnicas.
